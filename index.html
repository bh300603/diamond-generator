Diamond Generator

Description:

This is a Javascript based generative web application. Use the controls on the right to manipulate the diamonds.

Parameters:

  • numpoints: This controls the number of points in each diamond.
  • stroke: This controls the weight of the stroke in each diamond.
  • r: This controls the amount of red in each diamond.
    g: This controls the amount of green in each diamond.
    b: This controls the amount of blue in each diamond.
    alpha: This controls the opacity of each diamond.
    color: This controls the color of the stroke.
    bColor: This controls the color of the background.
